Directions to access product or any other additional information

For the application to work as it should, the user should have a uArm (robotic arm) connected to the power and to the computer, and then open the application.

However, if you don't have the uArm, then you can click on the following LINK so you can use the application without having a uArm and then click "Connect" without a connection, instead of choosing a port.

Additionally, the Email button works only if you have "run" the steps, have exercises and a log. I have provided a list of exercises and steps that you can open with the application located in the folder called "Library".
